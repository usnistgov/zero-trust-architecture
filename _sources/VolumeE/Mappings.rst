Mappings
===========

.. include:: /_publication_note.rst

Each mapping is available online as an Excel file. Four categories of mappings are available: CSF 1.1 subcategory mappings, CSF 2.0 subcategory mappings, NIST SP 800-53 control mappings, and EO 14028 security measure mappings. Within each category of mapping, there is both a general mapping from the ZTA reference design logical components to the document being mapped to (i.e., CSF, SP 800-53, or EO 14028), as well as a set of collaborator-specific mappings from the ZTA technology component capabilities that are included in one or more project builds to the document being mapped to (CSF, SP 800-53, or EO 14028). Table 1 provides links to each of the mappings.
